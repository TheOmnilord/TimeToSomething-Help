# Time To Something — Help &amp; Setup

This repository hosts the public help and setup page for the **Time To Something** Alexa skill at:

**https://theomnilord.github.io/TimeToSomething-Help/**

The page covers what the skill does, how to grant the Reminders permission, the recommended
multi-room setup (a device group for reminders in the Alexa app), and the full list of voice
commands — in English and German.

The skill itself lives in a separate repository.
